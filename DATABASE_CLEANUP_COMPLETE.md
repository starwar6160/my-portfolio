## 🕐 时间显示说明

### 数据库存储
- 数据库始终以 **UTC** 存储
- ISO 8601 格式: `2026-04-13T01:04:12Z`

### 仪表板显示
- 自动转换为 **JST** (UTC+9)
- 格式: `2026/04/13 10:04:12 (JST)`
- 明确标注时区，避免混淆

### 为什么这样做？
1. **存储 UTC**: 国际标准，无需夏令时调整
2. **显示 JST**: 方便您查看（日本本地时间）
3. **标注清楚**: 避免 9 小时时差造成的误解

## 🔧 技术实现

### 时间转换代码
```javascript
// 从数据库获取的 UTC 时间
const utcTime = '2026-04-13T01:04:12';

// 转换为 JST 显示
const jstTime = new Date(utcTime + 'Z').toLocaleString('ja-JP', {
  timeZone: 'Asia/Tokyo',
  year: 'numeric',
  month: '2-digit',
  day: '2-digit',
  hour: '2-digit',
  minute: '2-digit',
  second: '2-digit',
  hour12: false
});

// 结果: "2026/04/13 10:04:12 (JST)"
```
