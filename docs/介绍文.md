【複雑システム向けシニアエンジニアのご紹介】

お疲れ様です。
複雑系・長期運用・Legacy刷新・Infra・Embedded・高信頼性領域に強みを持つシニアエンジニアをご提案します。

本人の強みは、単なる標準業務開発ではなく、複雑なシステムを読み解き、問題を構造化して、再現・切り分け・修正まで持っていくことです。
特に、以下のような領域と相性が良いです。

- Automotive / 車載
- Infra / Platform / SRE
- 金融システム / Legacy modernization
- Embedded / Edge systems
- Security / Protocol analysis
- 分散バックエンド / ミッションクリティカルシステム

■ 候補者概要
- 実務経験23年以上
- C/C++、Go、Python を中心に、Linux、shell、デバッグ、移行、障害解析に強み
- ドキュメントが薄い環境でも、コード・ログ・実行挙動から構造を把握し、短期間で実務に入れる
- 仕様の曖昧な現場でも、論点整理と再現条件の特定から進められる
