## 📊 新表格列详解

### 1. Start Time（开始时间）
```
格式: 2026/04/13 14:12 (JST)
显示: 会话开始时间（日本时间）
简洁格式: 只显示到分钟，不显示秒
```

### 2. Duration（持续时间）🆕
```
格式: 1m 30s 或 45s 或 0s
含义: 从会话开始到最后一次请求的时间

示例:
- 0s: 只请求了一次（可能是爬虫预览）
- 5s: 快速浏览
- 1m 30s: 正常浏览
- 5m+: 深度阅读（高质量访客）
```

### 3. IP（访客 IP）
```
格式: 202.213.135.32
样式: monospace 字体，小字号
限制: 最多显示 20 个字符
```

### 4. Landing Page（落地页）
```
格式: /ja/ 或 / 或 /cases/
含义: 访客首次访问的页面
限制: 最多显示 30 个字符
悬停: 显示完整路径
```

### 5. Source（来源）
```
彩色标签:
- 灰色: Direct（直接访问）
- 蓝色: LinkedIn
- 绿色: Google
- 浅蓝: Twitter
- 深蓝: Facebook
- 紫色: Other
```

### 6. Activity（活动摘要）🆕
```
智能分类:

1. Quick view（灰色）
   - 页面浏览 <= 1 且 hits <= 5
   - 表示: 快速查看或爬虫预览
   - 示例: "Quick view"

2. Deep browse（绿色，加粗）
   - 页面浏览 >= 3
   - 表示: 深度浏览，高质量访客
   - 示例: "Deep browse (5 pages)"

3. Active（蓝色）
   - hits >= 10
   - 表示: 活跃用户，加载了很多资源
   - 示例: "Active (25 requests)"

4. 普通描述（灰色）
   - 其他情况
   - 显示页面数和请求数
   - 示例: "2 pages, 8 hits"
```

### 7. Type（访客类型）
```
绿色标签: New（首次访问）
蓝色标签: Returning（回访）
```
